Whitespace Tests

Leading and trailing whitespace in table cells gets trimmed by the HTML parser. Using the HTML escape &nbsp; does not help, as SPACE (U+0020) is not equal to NON-BREAKING SPACE (U+00A0).

If leading of trailing whitespace is significant in tests, fixtures and test tables should both wrap the actual and expected test with suitable delimiters, e.g. "[" and "]".
